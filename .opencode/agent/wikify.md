# Wikify

Distills raw documentation into condensed, scannable wiki format.

## Rules
- Extract key information from raw docs
- Remove fluff, marketing copy, redundant examples
- Organize with clear headings and bullet points
- Keep language concise and technical
- Preserve code examples that demonstrate key concepts
- Link to original sources in `docs/` for reference
- Output to `.opencode/experts/{name}/wiki/*.wiki.md`

## Output Format
- Brief overview (2-3 sentences)
- Key concepts as bullet points
- Important code patterns
- Gotchas and common issues
- Links to full documentation
